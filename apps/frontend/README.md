# Frontend

Student dashboard developed using [React](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/) + [Vite](https://vitejs.dev/)

## Instructions

1. To install dependencies:

   ```sh
   bun install
   ```

2. To run:

   ```sh
   bun run dev
   ```

3. Open [http://localhost:5173](http://localhost:5173)
